# Projeto Lista de Tarefas - Next.js 15

Projeto desenvolvido para praticar testes unitários em uma aplicação Next.js 15 com App Router e TypeScript.

## Funcionalidades

- Exibição de uma lista de tarefas simulada
- Adição de novas tarefas por formulário controlado
- Contagem de tarefas com hook personalizado
- Testes unitários com Jest e Testing Library

## Tecnologias usadas

- Next.js 15
- TypeScript
- React
- Jest
- Testing Library
